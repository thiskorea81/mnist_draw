# MNIST 스타일 손글씨 수집 도구

이 저장소는 **학생들과 함께 MNIST 스타일 숫자 데이터셋을 직접 만들어보고, 다양한 이미지 증강(Augmentation)을 체험**해 볼 수 있는 간단한 웹 도구입니다.

Vue.js CDN + 순수 HTML/JS만으로 구성되어 있어서, **파일 하나만 열면 바로 사용**할 수 있습니다.

https://mnistdraw.netlify.app/

---

## 1. 주요 기능

### ✅ 손글씨 숫자 수집

- 숫자 **0 ~ 9**까지 선택해서 연습
- 하나의 캔버스에 **10 × 5 그리드(총 50칸)** 제공  
  → 각 칸은 내부 해상도 **30 × 30 픽셀**
- 화면에서는 크게 보이도록 **1200 × 600**으로 확대 렌더링  
  (학생들이 편하게 손으로 쓸 수 있음)
- 각 칸에 같은 숫자를 50번 작성한 뒤, 버튼 한 번으로  
  → **28 × 28 이미지 50장으로 자동 분할 및 저장**

### ✅ 28×28 원본 이미지 생성

- 버튼: **`50칸 → 28×28 원본 50장 저장`**
- 내부 동작:
  - 300×150 캔버스를 10×5 칸으로 나눔
  - 각 칸(30×30)을 **28×28**로 리샘플링
  - 숫자별로 메모리에 **오프스크린 캔버스 형태로 50장 저장**
- 화면 아래에 **원본 예시 이미지(최대 10장)**를 미리보기로 표시

> 이 단계는 실제 파일로 저장되는 것이 아니라,  
> 브라우저 메모리에 28×28 캔버스로 보관되는 단계입니다.

### ✅ 데이터셋용 증강 + ZIP 다운로드 (1000장)

- 버튼: **`(원본+변환) 1000장 생성 & ZIP 다운로드`**
- 동작:
  - 현재 선택된 숫자에 대해
    - 원본 28×28 이미지: 최대 50장
    - 부족한 만큼 랜덤 증강을 적용해 **총 1000장**으로 채움
  - 파일 이름 형식:
    - `img_숫자_001.jpg` ~ `img_숫자_050.jpg` → 원본
    - `img_숫자_051.jpg` ~ `img_숫자_1000.jpg` → 증강 이미지
  - ZIP 파일 이름:
    - `digit_숫자_1000_images.zip`
- 화면에서는 랜덤으로 생성된 증강 이미지 중 **최대 10장**을 미리보기로 보여주며,
  각 이미지 아래에 **적용된 증강 파라미터**가 표시됩니다.

## 2. 증강(Augmentation) 방식

랜덤 증강 및 설명용 샘플에서 사용하는 변환은 다음과 같습니다.

### 🔁 회전 (Rotation)

- 범위: **-20° ~ +20°**
- 정보 표기 예:
  - `Rotate: 12.3°`

### ↔️ / ↕️ 이동 (Shift)

- X, Y 방향으로 작은 평행 이동
- 범위: **-3 ~ +3 픽셀** (설명용 샘플에서는 약간 더 크게 ±8까지)
- 정보 표기 예:
  - `Shift: (-1.5, 2.3)`

### 🔍 크기 조절 (Scale)

- 약간 확대하거나 축소
- 범위: **0.9 ~ 1.1** (설명용 샘플 일부는 0.8 ~ 1.2)
- 정보 표기 예:
  - `Scale: 1.04`

### 🎲 노이즈 (Noise)

- 각 픽셀의 RGB 값에 작은 랜덤값을 더해서 “살짝 흔들린” 느낌을 줌
- 데이터셋용 랜덤 증강:
  - 약 **70% 확률로 노이즈 ON**
- 정보 표기 예:
  - `Noise: ON` / `Noise: OFF`

## 3. 증강 방식별 설명용 샘플

수업에서 증강을 설명하기 위한 **시각적 예시**도 자동으로 생성할 수 있습니다.

- 버튼: **`증강 방식별 설명 샘플 생성`**
- 기준:
  - 현재 숫자의 28×28 원본 중 **첫 번째 이미지를 기준**으로 사용
- 생성 내용: 각 증강 방식마다 **5개씩**
  - `회전만 적용` 5개
  - `이동만 적용` 5개
  - `크기 조절만 적용` 5개
  - `노이즈만 적용` 5개
- 화면에 다음과 같이 구역별로 표시:
  - 회전 (Rotation)
  - 이동 (Shift)
  - 크기 조절 (Scale)
  - 노이즈 (Noise)
- 각 이미지 아래에는 다음 정보가 함께 표시됩니다.
  - `Rotate: ...`
  - `Shift: ...`
  - `Scale: ...`
  - `Noise: ...`

이를 통해 학생들이 **“이 변환이 실제 이미지에 어떤 영향을 미치는지”** 직관적으로 비교·관찰할 수 있습니다.

## 4. 사용 방법

### 4.1 준비물

- `mnist_draw.html` 파일 1개
- 인터넷 연결 (Vue, JSZip, FileSaver CDN 사용)

### 4.2 실행

1. `mnist_draw.html` 파일을 브라우저(Chrome 권장)에서 연다.
2. 상단의 숫자 버튼 중 **수집할 숫자(0~9)**를 선택한다.
3. 큰 캔버스(10×5 칸)에 같은 숫자를 50번 손으로 쓴다.
   - 펜처럼 마우스를 드래그해서 작성
   - `전체 지우기` 버튼으로 다시 시작 가능
4. 다 그렸으면:
   - **`50칸 → 28×28 원본 50장 저장`** 버튼 클릭  
     → 메모리에 28×28 이미지 50장 저장 + 원본 샘플 미리보기
5. 데이터셋용 이미지가 필요하면:
   - **`(원본+변환) 1000장 생성 & ZIP 다운로드`** 클릭  
     → 1000장 이미지 ZIP 파일 다운로드
6. 수업 설명용 증강 예시가 필요하면:
   - **`증강 방식별 설명 샘플 생성`** 클릭  
     → 아래에 증강 방식별(회전/이동/크기/노이즈) 5개씩 예시 표시

### 4.3 숫자별 데이터 초기화

- 현재 선택된 숫자에 대해 저장된 원본 및 증강 샘플 정보를 삭제하려면:
  - **`현재 숫자 데이터 초기화`** 버튼 클릭
  - 해당 숫자의 원본 및 샘플들이 모두 초기화됨

## 5. 파일 구조

이 프로젝트는 단일 HTML 파일로 구성되어 있으며, 기본 구조는 다음과 같습니다.

- **HTML `<head>`**
  - Vue.js CDN (`https://unpkg.com/vue@3/dist/vue.global.prod.js`)
  - JSZip (`https://cdn.jsdelivr.net/npm/jszip@3/dist/jszip.min.js`)
  - FileSaver (`https://cdn.jsdelivr.net/npm/file-saver@2/dist/FileSaver.min.js`)
  - 간단한 CSS 스타일(레이아웃, 버튼, 샘플 영역 등)

- **HTML `<body>`**
  - `#app` 루트 엘리먼트
  - 숫자 선택 버튼, 기능 버튼, 캔버스, 샘플 영역 등의 UI

- **Vue 앱 스크립트**
  - `digitStore`: 숫자별 28×28 원본 캔버스 저장소
  - 그리기 이벤트(poinerdown/move/up) 처리
  - 10×5 그리드 + 28×28 슬라이싱 로직
  - 증강 함수:
    - `applyAugmentation(baseCanvas, preset)`
    - `augment28x28(baseCanvas)` (랜덤)
  - ZIP 생성 및 다운로드 (JSZip + FileSaver)
  - 증강 설명용 샘플 생성 로직

## 6. 수업 활용 아이디어

- **MNIST 데이터셋 소개 + 직접 만들어보기**
  - 학생들이 직접 손글씨를 쓰고, 28×28 흑백 이미지로 변환되는 과정을 체험
- **데이터 증강의 필요성과 효과 설명**
  - 원본 데이터만 있을 때와, 증강된 데이터를 포함했을 때의 차이를 프로젝트로 비교
- **증강 파라미터 토론**
  - 회전 각도, 이동 거리, 스케일, 노이즈 강도 등을 조절해 보면서
  - “모델 성능에 어떤 영향을 줄까?” 라는 질문으로 토론 유도
- **모델 학습 실습과 연계**
  - 생성된 이미지 파일을 파이썬/케라스/파이토치 등으로 불러와
  - 간단한 CNN 모델 학습까지 확장 가능
